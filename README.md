# 自动签到脚本 - 基本操作

1.  点击`settings > Secrets`;
2.  点击`New repository secret`;
3.  在`Name`输入`JUEJIN`或`TIEBA`;
4.  在`Value`输入从掘金或者贴吧获取到的`cookie`;
5.  点击`Actions`开启流;
